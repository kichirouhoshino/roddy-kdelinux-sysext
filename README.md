# My Personal KDE Linux System Extensions

This repository contains my personal scripts for installing system extensions (sysexts).

## ⚠️ Important Notes

- **The scripts in this repository were vibecoded**. I have obviously verified and tested the scripts myself, and I am fully aware that not everyone is a fan of AI-generated scripts due to their fragility. If you're against these kinds of scripts, please don't use it. Please review the code carefully and use it with caution!
- These scripts are tailored for KDE Linux, specifically its current Arch Linux base. Once they switch to the BuildStream base, these scripts will likely need to be updated.
- Before running my installer for Cloudflare Warp, make sure to run setup-resolved.sh first to prevent networking issues, especially with Tailscale.

Don't expect frequent updates. I will only update the packages when needed or when I have the time to.
